음식정보

김치볶음밥

반숙 계란후라이를 얹은 맛있는 김치볶음밥

재료

  • 배추김치300g

  • 돼지고기(삼겹살 또는 목살) 200g

  • 양파 ½개

  • 밥 4컵

  • 설탕 ½큰술

  • 참기름 1큰술

  • 식용유 2큰술

  • 쪽파 3개

  • 계란후라이 4개

조리순서

  1. 배추김치, 돼지고기,양파를 잘게 썬다.

  2. 팬에 식용유를 두르고 센 불에서 양파와 돼지고기를 볶다가
    재료가 익으면 중간 불로 줄이고 김치와 설탕을 넣고 숨이
    죽을 때까지 약 3분간 더 볶는다.

  3. 밥을 넣고 센 불에서 볶다가 불을 끄고 참기름을 두른 후 잘 섞는다.

  4. 계란후라이를 김치볶음밥에 올리고 다진 쪽파를 뿌려 낸다.
